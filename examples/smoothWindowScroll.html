Simple usage:
new ScrollFrames(window);
Usage with options:
new ScrollFrames("#cross", {
  • duration: 3000, //default is 1200
  • ease: t => Math.min(1, 1.001 - Math.pow(2, -10 * t)), //this is the default function, but could be changed to custom one
  • multiply: 3, //mouse wheel multiplier, default is 1
});
